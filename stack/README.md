# Monitoring Stack

This directory will contain Docker Compose files for the local development
monitoring stack:

- **Seq** - structured log aggregation
- **OpenTelemetry Collector** - trace collection and export
- **Prometheus** - metrics scraping and storage
- **Grafana** - dashboards and visualization

See [Issue #7](https://github.com/Bravyr/bravyr-obs/issues/7) for implementation details.
